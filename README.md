Gerador de Currículo em PHP

Este é o projeto prático (APO) para a disciplina de Fundamentos de Programação para Internet. O objetivo é criar uma aplicação web que funciona como um gerador de currículos.

O usuário preenche um formulário com seus dados pessoais, formação e experiências, e a aplicação gera uma página de currículo formatada pronta para impressão.

Tecnologias Utilizadas

Este projeto foi desenvolvido utilizando as seguintes tecnologias, conforme solicitado:

Backend: PHP

Frontend: HTML, CSS e JavaScript

Framework CSS: Bootstrap (para design e layout)

Ambiente de Servidor: Xampp (Apache)

Controle de Versão: Git e GitHub

Funcionalidades Implementadas

Formulário de entrada de dados (HTML/Bootstrap).

Cálculo automático de idade (JavaScript).

Adição dinâmica de campos de "Experiência Profissional" (JavaScript).

Processamento e formatação dos dados no backend (PHP).

Geração de uma página de currículo para impressão.

Função “Baixar Currículo” (usando window.print()).

Como Executar o Projeto

Certifique-se de ter o Xampp
 instalado e o módulo Apache em execução.

Clone ou baixe este repositório para dentro da pasta htdocs do seu Xampp (ex: C:/xampp/htdocs/).

Abra seu navegador e acesse: http://localhost/gerador-curriculo/.
