#  Gerador de Currículo em PHP

[cite_start]Este é o projeto prático (APO) para a disciplina de **Fundamentos de Programação para Internet**[cite: 2]. O objetivo é criar uma aplicação web que funciona como um gerador de currículos.

[cite_start]O usuário preenche um formulário com seus dados pessoais, formação e experiências, e a aplicação gera uma página de currículo formatada pronta para impressão[cite: 19, 51].

##  Tecnologias Utilizadas

Este projeto foi desenvolvido utilizando as seguintes tecnologias, conforme solicitado:

* [cite_start]**Backend:** PHP [cite: 34, 41]
* [cite_start]**Frontend:** HTML, CSS e JavaScript [cite: 35]
* [cite_start]**Framework CSS:** Bootstrap (para design e layout) [cite: 26, 35]
* [cite_start]**Ambiente de Servidor:** Xampp (Apache) [cite: 22]
* [cite_start]**Controle de Versão:** Git e GitHub [cite: 21]

##  Funcionalidades Implementadas

* Formulário de entrada de dados (HTML/Bootstrap).
* [cite_start]Cálculo automático de idade (JavaScript)[cite: 38].
* [cite_start]Adição dinâmica de campos de "Experiência Profissional" (JavaScript)[cite: 39, 40].
* [cite_start]Processamento e formatação dos dados no backend (PHP)[cite: 41].
* [cite_start]Geração de uma página de currículo para impressão[cite: 51].
* [cite_start]Função "Baixar Currículo" (usando `window.print()`)[cite: 42].

##  Como Executar o Projeto

1.  [cite_start]Certifique-se de ter o [Xampp](https://www.apachefriends.org/pt_br/index.html) instalado e o módulo **Apache** em execução[cite: 22].
2.  Clone ou baixe este repositório para dentro da pasta `htdocs` do seu Xampp (ex: `C:/xampp/htdocs/`).
3.  Abra seu navegador e acesse: `http://localhost/gerador-curriculo/`.
